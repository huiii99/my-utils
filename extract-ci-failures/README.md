# 🧪 CI Failures → azdev test

Paste CI failure logs and instantly get `azdev test` commands grouped by a configurable batch size.

## Features

- Parses `FAILED ...::test_name` lines from pytest output
- Groups test names into `azdev test` commands (default 6 per line, configurable)
- Copy individual lines or all at once
- Auto-converts on paste

## Usage

Open `index.html` in your browser:

```
start index.html
```
