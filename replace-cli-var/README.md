# 🔧 CLI Variable Filler

Paste a CLI command containing `{variables}` and interactively fill in values. The resolved command updates in real time and can be copied with one click.

## Features

- Auto-detects `{placeholder}` variables from your command template
- Live preview with color-coded filled / unfilled variables
- **Multiline concatenation mode** — strips surrounding quotes and joins multi-line commands into one
- One-click copy to clipboard

## Usage

Open `index.html` in your browser:

```
start index.html
```
